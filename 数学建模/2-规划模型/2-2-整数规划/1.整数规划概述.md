>当变量限制为整数,则称为整数规划,目前主流的求解整数规划,往往只适用于整数线性规划,本章节主要介绍“0-1规划”和蒙特卡罗规划

## 一、 0-1规划
0-1规划是整数规划中的特殊情形。变量$x_j$仅取值0或1，有着如下约束$0\leq X_j \leq 1且x_j为整数$
### 1.相互排斥的约束条件
有着两种运输方式可供选择，但只能选择一种运输方式，或者用车运输，或者用船运输。用车运输的约束条件为$5x_1+4x_2 \leq 24$，用船运输的约束条件为$7x_1+3x_2 \leq 45$。因此有如下约束条件$$
5x_1+4x_2 \leq 24 或 7x_1+3x_2 \leq 45 $$为了统一问题，引入0-1变量$$
y=\begin{cases} 1,用车运输\\0,用船运输 \end{cases}$$则约束条件可以改写成，设M为无限大数：$$
\begin{cases} 5x_1+4x_2 \leq 24+yM \\ 7x_1+3x_2 \leq 45+(1-y)M \\y=0 或 1\end{cases}
$$
如果有m个相互排斥的约束条件，为保证m个约束条件仅有一个起作用，引入m个0-1变量$$y_i=\begin{cases}1,第i个约束起作用\\0,第i个约束不起作用，i=1,2,...,m\end{cases}$$
**代码示例：** 假设我们要最大化收益 $f = 10x_1 + 8x_2$，约束条件就是笔记里的“用车或用船”： $5x_1 + 4x_2 - My \le 24$ 和 $7x_1 + 3x_2 + My \le 45 + M$ （设极大值 $M = 1000$，变量顺序为 $x_1, x_2, y$）
```matlab
% 1. 目标函数求最大值，MATLAB默认求最小，所以系数加负号
f = [-10; -8; 0]; 

% 2. 不等式约束矩阵 A * x <= b
M = 1000;
A = [5, 4, -M; 
     7, 3,  M];
b = [24; 45 + M];

% 3. 指定整数变量 (第3个变量 y 是整数)
intcon = 3; 

% 4. 设置上下界 (x1, x2 >= 0; 0 <= y <= 1)
lb = [0; 0; 0];
ub = [inf; inf; 1]; 

% 5. 求解
[x, fval] = intlinprog(f, intcon, A, b, [], [], lb, ub);

disp('最优解为：');
disp(x);
```
### 2.关于固定费用的问题
在实际的生产或投资规划中，经常会遇到“固定费用”问题。比如：生产某种产品，只要开工（无论产量多少），就需要支付一笔固定的设备准备费或开机费（固定费用）；此外，还需要支付与产量成正比的生产费用（变动费用）。

假设生产第 $j$ 种产品的固定费用为 $K_j$，单位变动成本为 $c_j$，产量为 $x_j$。那么生产第 $j$ 种产品的总费用 $C_j$ 可以表示为：
$$C_j = \begin{cases} K_j + c_j x_j, & x_j > 0 \\ 0, & x_j = 0 \end{cases}$$
显然，这里的目标函数或者约束条件包含了非线性的分段函数，直接求解非常困难。为了将其转化为线性整数规划，我们引入 0-1 变量 $y_j$ 来表示“是否生产”：
$$y_j = \begin{cases} 1, & \text{当生产该产品时 (即 } x_j > 0) \\ 0, & \text{当不生产该产品时 (即 } x_j = 0) \end{cases}$$
利用大 M 法（设 $M$ 为一个足够大的正数），我们可以将总费用线性化，并建立如下约束：
$$C_j = K_j y_j + c_j x_j$$
$$\begin{cases} x_j \leq M y_j \\ x_j \geq 0 \\ y_j \in \{0, 1\} \end{cases}$$
**模型逻辑解析：**

- 当 $y_j = 0$ 时，根据约束 $x_j \leq 0$ 且 $x_j \geq 0$，强制使得 $x_j = 0$，此时费用 $C_j = 0$。
- 当 $y_j = 1$ 时，约束变为 $x_j \leq M$（由于 $M$ 极大，相当于对 $x_j$ 的上限没有限制），此时费用 $C_j = K_j + c_j x_j$，完美贴合了固定费用的逻辑。

### 3. 指派问题的数学模型

**指派问题 (Assignment Problem)** 是 0-1 规划中极其经典的一类问题。它的标准描述是：有 $n$ 个人要去完成 $n$ 项任务，每个人只能完成一项任务，每项任务也只能交由一个人去完成。已知第 $i$ 个人完成第 $j$ 项任务所需的成本（或时间）为 $c_{ij}$，如何分配任务才能使总成本最小？

首先，引入 0-1 决策变量 $x_{ij}$：
$$x_{ij} = \begin{cases} 1, & \text{指派第 } i \text{ 个人去完成第 } j \text{ 项任务} \\ 0, & \text{不指派第 } i \text{ 个人去完成第 } j \text{ 项任务} \end{cases} \quad (i, j = 1, 2, ..., n)$$
**完整的数学模型如下：**

**目标函数 (最小化总成本)：**
$$\min Z = \sum_{i=1}^{n} \sum_{j=1}^{n} c_{ij} x_{ij}$$
**约束条件：**
$$\begin{cases} \sum_{j=1}^{n} x_{ij} = 1, & i = 1, 2, ..., n \quad \text{(每个人只能分配到一项任务)} \\ \sum_{i=1}^{n} x_{ij} = 1, & j = 1, 2, ..., n \quad \text{(每项任务只能由一个人完成)} \\ x_{ij} \in \{0, 1\}, & i,j = 1, 2, ..., n \quad \text{(决策变量只能取 0 或 1)} \end{cases}$$
指派问题因为其特殊的约束矩阵结构（系数矩阵完全由 0 和 1 组成），通常可以使用专门的、更高效的**匈牙利算法 (Hungarian Algorithm)** 来求解，而不需要动用通用的分支定界法或蒙特卡洛法。
## 二、蒙特卡洛法
蒙特卡洛法是基于大量事件的统计结果来实现一些确定性问题的计算。使用其方法，必须使用大量random_seed(n)，**通常用于寻找**“满意解”**或**“近似最优解”。
### 1.数学原理
假设一个极其庞大的解空间中有上百亿种组合，但其中有 **$0.1\%$** 的解是符合我们要求的“优质解”（或者叫近似最优解）。
- 每次随机采样，抽到“劣质解”的概率是 $0.999$。
- 连续采样 $N$ 次，**全部完美避开**优质解的概率是 $0.999^N$。

如果我们希望有 **$99\%$** 的把握找到至少一个优质解，只需要满足：
$$0.999^N \leq 0.01$$
解这个对数方程：
$$N \geq \frac{\ln(0.01)}{\ln(0.999)} \approx 4602$$
matlab公式如下
```matlab
N = 10000; % 设定采样次数（掷骰子一万次）
best_fval = inf; % 初始化最优解（求最小化问题，先设为无穷大）
best_x = [];

for i = 1:N
    % 1. 随机生成一组 0-1 变量 (假设有 10 个 0-1 变量)
    random_x = randi([0, 1], 10, 1); 
    
    % 2. 检查是否满足所有约束条件
    if (满足约束条件) % 这里需要你写出具体的判断逻辑，比如 A*random_x <= b
        % 3. 计算目标函数值
        current_fval = 计算收益(random_x); 
        
        % 4. 如果比历史最优解好，就更新记录
        if current_fval < best_fval
            best_fval = current_fval;
            best_x = random_x;
        end
    end
end
```
