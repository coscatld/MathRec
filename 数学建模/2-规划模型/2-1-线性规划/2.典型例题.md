## 投资问题
### 一、abstract
市场上有n种资产$s_i(i=1,2,...,n)$，用数额为M（本题取M=1万）的资金作为一个时期的投资
这n种资产在这一时期内购买 $s_i$ 的平均收益率为 $r_i$，风险损失率为 $q_i$
投资越分散总的风险越少，总体风险用投资的 $s_i$ 中最大的风险来度量
购买 $s_i$ 时要付交易费，费率为 $p_i$ 
当购买额不超过给定值 $u_i$ 时, 交易费按购买 $u_i$ 计算
同期银行存款利率是 $r_0=5\%$ 且无交易费无风险，n = 4时相关数据如下表：
![697](img/img_1.png)
参数定义
- $s_i(i=1,2,...,n)$：资产
- $p_i$ ：购买$s_i$的交易费率
- $u_i$：最小购买额
- $r_i$：购买 $s_i$ 的平均收益率
- $q_i$：购买 $s_i$ 的风险损失率
### 二、problem
给该公司设计一种投资组合方案，用给定的资金M，有选择地购买若干种资产或存银行生息，使净收益尽可能大，总体风险尽可能小
### 三、解题过程
#### 1.Basic assumptions
- 由于投资的数额M相当大，而题目设定的定额$u_i$相对于M很小，$p_i u_i$更小，因此假设每一笔交易额$x_i$都大于对应的定额$u_i$
#### 2.建立数学模型的核心步骤
##### 1） 设定决策变量 (Decision Variables)

首先，我们需要明确要“决定”什么。在这个问题中，我们要决定的是分配给各个资产的资金具体是多少。

- 设 $x_0$ 为存入银行的资金。
- 设 $x_i$ 为购买资产 $s_i$ 的资金（$i = 1, 2, 3, 4$）。
##### 2） 建立目标函数 (Objective Functions)

题目要求“净收益尽可能大，总体风险尽可能小”，这是一个典型的**多目标优化问题**。我们需要把这两个目标用公式表达出来。

- **目标1：净收益最大化 ($Q$)**
    投资组合的总净收益 = 银行利息 + 各项资产收益 - 各项资产交易费。
    $$\max Q = r_0 x_0 + \sum_{i=1}^{4} (r_i - p_i)x_i$$
- **目标2：总体风险最小化 ($R$)**
    题目中明确指出：“总体风险用投资的 $s_i$ 中最大的风险来度量”，且银行存款无风险。
    $$\min R = \max_{1 \le i \le 4} \{q_i x_i\}$$
##### 3）确立约束条件 (Constraints)

资金是有限的，且投资金额不能为负数。
- **资金总额限制：** 所有的投资额加起来必须等于总资金 $M$。
    $$x_0 + x_1 + x_2 + x_3 + x_4 = M$$
- **非负限制：** 每一项投资金额都必须大于等于0。
    $$x_i \ge 0 \quad (i = 0, 1, 2, 3, 4)$$
##### 4）模型转化 (Model Transformation)

原问题同时要求“净收益尽可能大”和“总体风险尽可能小”，属于典型的多目标优化问题。为便于求解，本文采用**固定风险水平下最大化收益的方法**，即预先设定投资者能够承受的最大风险水平 $a$，在风险不超过该水平的条件下，使投资组合的净收益最大。

根据题意，投资组合的总体风险由各项风险损失中的最大值衡量，因此定义总体风险率为
$$  R=\max_{1\leq i\leq 4}\left\{\frac{q_i x_i}{M}\right\}
$$
设投资者能够承受的最大风险水平为 $a$，则应满足$R\leq a$，即
$$\max_{1\leq i\leq 4}\left\{\frac{q_i x_i}{M}\right\}\leq a$$
最大值不超过 $a$，等价于每一项均不超过 $a$，因此有
$$  
\frac{q_i x_i}{M}\leq a,\quad i=1,2,3,4  
$$
于是原多目标优化问题可以转化为如下参数线性规划模型：
$$  
\max Q=r_0x_0+\sum_{i=1}^{4}(r_i-p_i)x_i  
$$
约束条件为
$$  
\begin{cases}  
\dfrac{q_i x_i}{M}\leq a,\quad i=1,2,3,4\\ 
\displaystyle\sum_{i=0}^{4}x_i=M\\
x_i\geq0,\quad i=0,1,2,3,4  
\end{cases}  
$$

其中，$a$ 为投资者设定的最大可承受风险水平。通过改变 $a$ 的取值，可以分别求得不同风险水平下的最大净收益 $Q$ 及对应的最优投资组合 $x_i$。

因此，可以取一系列不同的风险水平 $a$，利用线性规划方法逐一求解，并绘制风险—收益曲线。该曲线反映投资组合风险与最优收益之间的关系，从而为公司根据自身风险承受能力选择投资方案提供依据。
#### 3.代入数据的线性规划模型
**1）目标函数（最大化净收益 $Q$）：**

$$\max Q = 0.05x_0 + 0.27x_1 + 0.19x_2 + 0.185x_3 + 0.185x_4$$

**2）约束条件（s.t.）：**

$$\begin{cases} \frac{0.025 x_1}{10000} \le a \\ \frac{0.015 x_2}{10000} \le a \\ \frac{0.055 x_3}{10000} \le a \\ \frac{0.026 x_4}{10000} \le a \\ x_0 + x_1 + x_2 + x_3 + x_4 = 10000 \\ x_0, x_1, x_2, x_3, x_4 \ge 0 \end{cases}$$
代码的逻辑是：让风险界限 $a$ 从 $0$ 慢慢增加到 $3\%$（因为当风险达到 $2.5\%$ 时，资金可以全投收益最高的 $s_1$，收益就见顶了），算出每一个 $a$ 对应的最大收益 $Q$，然后画图。
```python
import numpy as np
import matplotlib.pyplot as plt
from scipy.optimize import linprog

# 1. 初始化数据
M = 10000
# 净收益率 (s0, s1, s2, s3, s4)
net_returns = np.array([0.05, 0.27, 0.19, 0.185, 0.185])
# 风险率 (s0 是银行，风险为0)
risks = np.array([0, 0.025, 0.015, 0.055, 0.026])

# linprog 默认是求最小值，所以我们将目标函数系数取负
c = -net_returns

# 等式约束: x0 + x1 + x2 + x3 + x4 = 10000
A_eq = np.array([[1, 1, 1, 1, 1]])
b_eq = np.array([M])

# 准备记录结果的列表
a_values = np.linspace(0, 0.03, 100) # 风险从 0 到 3%
max_returns = []

# 2. 循环遍历不同的风险界限 a，求解线性规划
for a in a_values:
    # 不等式约束: q_i * x_i <= a * M
    # 因为 x0 无风险，只约束 x1 到 x4
    A_ub = np.zeros((4, 5))
    np.fill_diagonal(A_ub[:, 1:], risks[1:])
    b_ub = np.full(4, a * M)
    
    # 变量非负约束: x_i >= 0
    bounds = [(0, None) for _ in range(5)]
    
    # 求解
    res = linprog(c, A_ub=A_ub, b_ub=b_ub, A_eq=A_eq, b_eq=b_eq, bounds=bounds, method='highs')
    
    if res.success:
        max_returns.append(-res.fun) # 结果取反恢复正收益
    else:
        max_returns.append(0)

# 3. 绘制风险-收益曲线 (有效前沿)
plt.figure(figsize=(8, 5))
plt.plot(a_values * 100, max_returns, 'b-', linewidth=2)
plt.xlabel('最大可承受风险 a (%)')
plt.ylabel('最大净收益 Q (元)')
plt.title('投资组合：风险 - 收益曲线 (有效前沿)')
plt.grid(True, linestyle='--', alpha=0.7)
plt.show()
```
